# Este repositorio contiene el desarrollo del Proyecto Integrador 2 - Grupo 2, enfocado en el diseño e implementación de un sistema web para la gestión integral del ciclo de vida del colaborador en el área de Recursos Humanos.

El proyecto surge ante la necesidad de optimizar procesos que actualmente se realizan de forma manual y desorganizada, como el reclutamiento, seguimiento de postulantes y gestión de información del personal. Frente a esta problemática, se propone una solución tecnológica que permita centralizar la información, automatizar tareas y mejorar la trazabilidad de cada etapa del colaborador dentro de la organización.

Además, el sistema incorpora funcionalidades de analítica de flujo de personal, facilitando la generación de reportes e indicadores clave que apoyan la toma de decisiones estratégicas. La solución está desarrollada como una aplicación web utilizando tecnologías modernas, con el objetivo de ofrecer una plataforma eficiente, escalable y orientada a mejorar la gestión del talento humano.

Este proyecto refleja el trabajo colaborativo del equipo y la aplicación de metodologías ágiles, buenas prácticas de desarrollo y herramientas de control de versiones.
